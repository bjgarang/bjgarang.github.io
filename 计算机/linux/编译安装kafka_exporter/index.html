    附录：go get使用时的附加参数¶

    使用 go get 时可以配合附加参数显示更多的信息及实现特殊的下载和安装操作，详见下表所示。

    附加参数 备 注
    -v 显示操作流程的日志及信息，方便检查错误
    -u 下载丢失的包，但不会更新已经存在的包
    -d 只下载，不安装
    -insecure 允许使用不安全的 HTTP 方式进行下载操作
